
# Personal Expense Tracker (Python)

This is a simple Python command-line project to track daily expenses.

## Features:
- Add expenses with name, amount, category
- View all expenses
- Calculate total spending
- Data stored in JSON file

## Tech Used:
- Python
- File Handling (JSON)

## How to Run:
python main.py

## Note:
All data is saved in expenses.json file.
